分类和回归的区别在于输出变量的类型。  
定量输出称为回归，或者说是连续变量预测。找到最有拟合。  
定性输出称为分类，或者说是离散变量预测。寻找决策边界。  
举个例子：  
预测明天的气温是多少度，这是一个回归任务。  
预测明天是阴、晴还是雨，就是一个分类任务。  
若我们欲预测的是离散值，例如"好瓜""坏瓜"，此类学习任务称为 "分类"。  
若欲预测的是连续值，例如西瓜的成熟度0.95 ,0.37,此类学习任务称为"回归"。
